\doxysubsection{Detailed Description}
Implementation of the SFML graphics library for handling arcade graphics. 

This class inherits from \doxylink{class_i_graphic}{IGraphic} and provides concrete implementations for SFML-\/based graphics handling. It includes methods for initializing graphics, drawing menus, managing events, updating graphics, handling sounds, and providing getters/setters for various properties.
