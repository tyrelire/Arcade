Detailed Description

Core class responsible for managing the main game loop and coordinating interactions between graphics and game logic.

This class orchestrates the main game loop, handles user input, and manages interactions between the graphics interface and game logic. It provides methods for running the game, updating menus, switching between graphics libraries and game implementations, and setting/getting various game parameters.
