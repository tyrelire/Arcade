Detailed Description

template<typename ClassContainer>
class Loader< ClassContainer >

Generic class for dynamically loading shared libraries and instantiating objects from them.

This class provides functionality for loading shared libraries dynamically and creating instances of objects from them. It supports loading libraries at runtime, swapping between loaded libraries, and obtaining paths to available libraries.

Template Parameters
ClassContainerThe class type to be instantiated from the loaded library.
